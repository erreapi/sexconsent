# SexConsent 💘

A beautiful, playful, and respectful React app for couples to communicate desires, boundaries, and discover new adventures together, featuring animated UI, language support, and a visually stunning passion meter.

---

## ✨ Features
- Multi-language support (EN/IT/ES/FR)
- Gorgeous animated UI with custom backgrounds
- Fully animated Passion Meter (flames!)
- Random flirty prompt generator
- Responsive, mobile-friendly, and easy to extend

---

## 🛠 Installation & Setup

1. **Clone the repository**
   ```bash
   git clone https://github.com/erreapi/sexconsent.git
   cd sexconsent
   ```

2. **Install required dependencies**
   ```bash
   npm install
   # or with yarn
yarn
   ```
   This installs:
   - React
   - Tailwind CSS
   - framer-motion (for beautiful animations)

3. **Install Tailwind CSS** (if not already installed)
   - [Follow the official guide](https://tailwindcss.com/docs/guides/create-react-app) for Create React App, Vite, or Next.js as appropriate. Ensure you have the correct Tailwind and `postcss` dependencies.

4. **Check framer-motion**
   ```bash
   npm install framer-motion
   # or
yarn add framer-motion
   ```

---

## 🚀 Running the App

```bash
npm start
# or
yarn start
```

Open [http://localhost:3000](http://localhost:3000) to view the app.

---

## 📄 Usage
- Use the language selector (top-right) to choose your language.
- For each activity card, use the animated buttons to indicate Yes / Maybe / No.
- Drag the Passion Meter for a beautiful flame animation and adjust the vibe!
- Use the Randomizer for a fun, flirty prompt.
- "Save Tonight’s Vibes" for a playful experience.

---

## ⚙️ Customization
- Categories and options can be edited directly in the `categories` array inside [`sexconsent.jsx`](sexconsent.jsx).
- Add more languages or extend translation as needed.
- Animation, flame SVGs, and background SVGs can be further customized to your taste.

---

## ❓ Troubleshooting
- Ensure all dependencies are installed.
- If Tailwind styles are not applying, re-check Tailwind setup with your React toolchain.
